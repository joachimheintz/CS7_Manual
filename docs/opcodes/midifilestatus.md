# midifilestatus
Returns the playback status of MIDI file input.

Returns the current playback status at k-rate, of the input MIDI file, 1 if file is playing, 0 if the end-of-the file has been reached.

## Syntax
``` csound-orc
ksig  midifilestatus
```
