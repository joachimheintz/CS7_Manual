# hypot
Euclidean distance function.

## Syntax
``` csound-orc
ires[] hypot iarg1[], iarg2[]
kres[] hypot karg1[], karg2[]
```

### Initialization

_iarg[]1/2_ -- the operands.

### Performance

_karg[]1/2_ -- the operands
