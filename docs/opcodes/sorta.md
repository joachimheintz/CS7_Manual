# sorta
Takes a numeric array (k or i-rate) and returns it sorted in ascending order.

## Syntax
``` csound-orc
k/i[] sorta k/i[]  (k- or i-arrays )
```

Here is an example of the sorta opcode. It uses the file [sorta.csd](../../examples/sorta.csd).
