# rewindscore
Rewinds the playback position of the current score performance.

## Syntax
``` csound-orc
rewindscore
```

## Examples

Here is an example of the _rewindscore_ opcode. It uses the file [rewindscore.csd](../../examples/rewindscore.csd).
